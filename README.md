> **"جسر رقمي يربط المغترب بقلب الوطن"**

Asmara.Store هي المنصة الرقمية الأولى من نوعها في إريتريا التي تدمج بين الخدمات اللوجستية (الطيران)، الاستثمارية (العقارات، السيارات، الإلكترونيات)، والمعرفية (الصحافة الرسمية) في واجهة ذكية واحدة. تم استلهام فلسفة التصميم من كبرى المتاجر العالمية مثل أمازون لتقديم تجربة مستخدم موثوقة، سريعة، ومتعددة اللغات.

## 🎯 الرؤية (Vision)

توفير منصة متكاملة وآمنة للمغتربين الإريتريين حول العالم، تربطهم بوطنهم من خلال:
- خدمات شراء وبيع موثوقة
- فرص استثمارية عقارية
- تتبع الشحنات والطلبات
- مصادر رسمية للأخبار والوثائق
- مجتمع آمن يحترم العادات والتقاليد الإريترية

## 🧩 الأقسام والخدمات

### 1. العقارات 🏠
- إضافة عقار (شقة، فيلا، أرض، مكتب)
- السعر، الوصف، الموقع (أسمرة، مصوع، كرن)
- رفع صور
- معرض عقارات

### 2. الإلكترونيات 📱
- هواتف، لابتوبات، أجهزة لوحية
- المواصفات والسعر
- معرض إلكترونيات

### 3. السيارات 🚗
- سيارات جديدة ومستعملة
- الماركة، الموديل، السعر
- معرض سيارات

### 4. الوظائف 💼
- دوام كامل، جزئي، عن بعد
- المسمى، الشركة، الراتب
- معرض وظائف

### 5. حجز الطيران ✈️
- رحلات من دبي، جدة، القاهرة، أديس أبابا
- اختيار التاريخ
- تأكيد الحجز

### 6. المؤسسات الرسمية 🏛️
- وزارة التجارة والصناعة (رابط COMESA)
- وزارة المعلومات (Shabait.com)
- مكتبة أسمرة (مركز التوثيق)

## 🛡️ نظام الوكيل والإشراف

### مهام الوكيل:

1. **مراجعة الإعلانات** قبل النشر
2. **منع المخالفات** (كلمات ممنوعة، محتوى غير لائق)
3. **معالجة البلاغات** من المستخدمين
4. **إدارة الكلمات الممنوعة** (إضافة/حذف)

### الكلمات الممنوعة تشمل:

| الفئة | أمثلة |
|-------|-------|
| 🔞 جنسية | سكس، عاري، نيك، زنا، porn |
| ☪️ دينية | كفر، إهانة دين، مسجد، كنيسة |
| 🤝 عنصرية | قبيلة، عنصرية، عرق، طائفي |
| 📏 عادات وتقاليد | شتيمة، سب، فاحشة، خمر |
| ⚠️ أخرى | روابط خارجية، أسعار غير واقعية |

### آلية الإبلاغ:

- كل إعلان يحتوي على زر بلاغ (🚨)
- المستخدم يكتب سبب البلاغ
- الوكيل يستعرض البلاغ في لوحة مخصصة
- إذا وصل عدد البلاغات إلى 3، يتم حظر الإعلان تلقائياً
