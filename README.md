## 📋 عن المشروع

**Asmara.Store** هو منصة رقمية متكاملة تهدف إلى:

- 🔗 ربط المغتربين الإريتريين بالخدمات داخل الوطن
- 🏠 تسهيل البحث عن العقارات والمنتجات والخدمات
- ✈️ حجز رحلات الطيران إلى أسمرة
- 💼 توفير فرص عمل للمواطنين
- 🛡️ الإشراف على المحتوى من خلال وكيل معتمد
