# 🇪🇷 Asmara.Store - المنصة الرقمية المتكاملة لإريتريا

![Version](https://img.shields.io/badge/version-2.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-web-orange)
![Languages](https://img.shields.io/badge/languages-3-gold)

> **"جسر رقمي يربط المغترب بقلب الوطن"**

Asmara.Store هي المنصة الرقمية الأولى من نوعها في إريتريا التي تدمج بين الخدمات اللوجستية (الطيران)، الاستثمارية (العقارات، السيارات، الإلكترونيات)، والمعرفية (الصحافة الرسمية) في واجهة ذكية واحدة. تم استلهام فلسفة التصميم من كبرى المتاجر العالمية مثل أمازون لتقديم تجربة مستخدم موثوقة، سريعة، ومتعددة اللغات.
